# Mini Katalog App

Flutter ile geliştirilmiş e-ticaret katalog uygulaması. Ürünleri keşfet, ara, detaylarını incele ve sepete ekle.

## Flutter Sürümü
Flutter 3.41.4

## Çalıştırma Adımları

```bash
flutter pub get
flutter run
```
